---
title: Semicolons
permalink: /docs/semicolons/
---

Semicolons are optional in Rose. A semicolon is equivalent to a newline character and can be used anytime a newline character can be used.

{% highlight ruby %}
    let x := 1
    let y := 2
{% endhighlight %}

is equivalent to

{% highlight ruby %}
    let x := 1 ; let y := 2
{% endhighlight %}
